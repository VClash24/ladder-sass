# Ladder
Ladder is a simple Sass mixin &amp; utility library.

## Installation & Usage

1.Install via NPM

```
npm install ladder-sass
```

2.Import base file into your main .scss stylesheet.
```
@import 'ladder-sass/ladder'
```
